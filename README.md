# firulais
Firulais es un potente cliente de API offline-first para desarrolladores que priorizan la privacidad. Organiza   peticiones jerárquicamente, gestiona entornos dinámicos y automatiza flujos con un motor JavaScript integrado. Ofrece   compatibilidad bidireccional perfecta para importar y exportar a Postman y Bruno, manteniendo tus datos 100% locales.
